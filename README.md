# 🚀 Portfolio 2.0

A professional, responsive, and modern personal portfolio website built to showcase my journey, projects, and technical expertise.

## 🔗 Live Demo
Check out the live site here: [https://harris8099.github.io/Portfolio-2.0/](https://harris8099.github.io/Portfolio-2.0/)

## ✨ Key Features
- **Modern UI/UX:** Clean design with a focus on readability and user experience.
- **Fully Responsive:** Optimized for all screen sizes, from mobile devices to large desktops.
- **Project Showcase:** Highlights key software development and cybersecurity projects.
- **Interactive Elements:** Smooth transitions and dynamic content using JavaScript.
- **Contact Integration:** Easy-to-find links for professional inquiries and social profiles.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for better accessibility and SEO.
- **CSS3:** Custom styling using modern layouts (Flexbox/Grid) and animations.
- **JavaScript (ES6+):** For interactive components and logic.
- **Google Fonts:** Enhanced typography for a professional look.
- **Font Awesome:** Iconography for social media and navigation.
